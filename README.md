# Cryox

## First time setup
```
git clone https://github.com/a56097634a/Cryox.git
cd Cryox
meteor create Cryox
mv Cryox/.meteor/ .
rm -rf Cryox/
```
```meteor add <dependency>``` for all dependencies

```meteor``` (to run server on localhost:3000)

### Dependencies
iron:router

sergeyt:typeahead
